R Computing Resources

  • Interactive R Basics Tutorial : This is an interactive tutorial that can be run by downloading the repository, installing and loading the learnr package, opening the project link, and using the code learnr::run_tutorial("Basic-R-Tutorial") . The tutorial includes information on creating vectors, dataframes, and performing basic arithmetic operations with vectors.
  • Video on Creating R Markdown Files : This is a YouTube video on the basics of creating R Markdown files.
  • Video on knitr::kable Package This is a YouTube video on creating tables using the knitr::kable package.
